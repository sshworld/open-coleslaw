# Open Dashboard

The Open Coleslaw dashboard runs at **http://localhost:35143** and starts automatically with the plugin.

Tell the user:

> The dashboard is running at http://localhost:35143
>
> It shows:
> - Real-time agent hierarchy (Orchestrator → Leaders → Workers)
> - Meeting progress and phases
> - Task delegation and completion with animated arrows
> - @Mention alerts for pending decisions
> - Event log and cost tracking
>
> Open it in your browser to see the Neon Ops Center!
